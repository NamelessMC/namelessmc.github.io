Current version: 0.4.0. Released on 13th July 2015.
NamelessMC is a website package written in PHP which aims to provide servers with a free and easy to use online community "hub".
Requirements

Communicate

Create a hub for your community with the forums, view latest news and your Twitter feed on the front page.

Integrate

View ingame player statistics and infractions on player profiles, and integrate your Buycraft or MinecraftMarket donation store with your website.

Customise

Switch between themes with the click of a button, and create your own custom pages.

Responsive

NamelessMC is built with Bootstrap, resulting in a fully responsive website.

Analyse

View statistics for your Minecraft servers, including server performance and player activity.

Open Source

NamelessMC is released under the MIT license, and all code is viewable on GitHub.
